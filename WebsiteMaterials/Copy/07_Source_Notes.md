## Steam Reference Used

The Steam store currently presents Knucklebones as:

- a dark tactical dice builder
- set on a snakes-and-ladders roguelike map
- focused on rolling, assigning, upgrading, and surviving

The copy in this folder keeps that core positioning but expands it into more website-friendly language and slightly broader atmospheric framing.
