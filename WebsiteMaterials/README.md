# Website Materials

This folder is a curated website and promo pack for `Knucklebones`.

It is intended for:

- the studio website
- store-page-adjacent promo use
- press-kit assembly
- social and newsletter support copy

## Folder Structure

- `Branding/Knucklebones/`
  - game logos, wordmark, and icons
- `Branding/IronQuill/`
  - currently available studio-brand asset found in the repo
- `Artwork/KeyArt/`
  - storefront-ready and hero-style promotional images
- `Artwork/Illustrations/`
  - selected in-game illustrations that represent tone, world, and encounters
- `Copy/`
  - drafted website and promo copy

## Notes

- The Knucklebones branding set was found in the repository and included directly.
- I did not find a full checked-in Iron Quill primary/secondary/logo pack in source asset folders.
- The currently available studio-mark asset included here is `IronQuillStudioMark.png`.
- Copy in `Copy/` draws from the current project documentation and the live Steam store description, then expands it into website-ready language.
